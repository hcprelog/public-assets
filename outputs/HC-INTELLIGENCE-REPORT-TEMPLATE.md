## HOW TO PRODUCE THIS REPORT (Monthly, ~4 hours)

### Step 1 — SAM.gov Search (1 hour)
Go to sam.gov/content/opportunities → Filters:
- Status: Active
- Set-Aside: SDVOSB + HUBZone (run separately)
- NAICS: 562211, 562910, 541614, 484110, 484121, 493110, 488510
- Posted: Last 30 days
- Due Date: Next 45 days

Export results → Pick the 10–15 most interesting → Run bid/no-bid score on each.

### Step 2 — Bid/No-Bid Scoring (1 hour)
Score each opportunity 1–10 on these factors:
- **NAICS match** (0-2 pts): Does H&C's registered NAICS match exactly?
- **Past performance** (0-2 pts): Does H&C have relevant PP to cite?
- **Competition level** (0-2 pts): How many likely competitors? (SDVOSB sole source = 2, open competition = 0)
- **Value range** (0-2 pts): Is it in H&C's sweet spot ($25K–$500K)?
- **Agency relationship** (0-2 pts): Have we worked with this agency or office before?

Score ≥7: Submit. Score 5–6: Submit if bandwidth allows. Score <5: Pass.

### Step 3 — Write the Report (1.5 hours)
Use the template below. Fill in real opportunities from your SAM.gov search.

### Step 4 — Send (30 min)
Mailchimp → Segment: GROWTH + PRIME READY subscribers → Send
